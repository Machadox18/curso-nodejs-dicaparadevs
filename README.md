🚀 API REST com Node.js, Express e EJS

Este projeto é uma API desenvolvida com Node.js e Express, capaz de realizar operações completas de CRUD de usuários. A aplicação permite criar, buscar, atualizar e deletar usuários, além de buscar usuários específicos pelo ID, retornando dados em JSON.

Também foi utilizado EJS para renderização de páginas no servidor, permitindo visualizar os usuários diretamente no navegador.

🔧 Tecnologias utilizadas:

Node.js

Express

MongoDB + Mongoose

EJS

Dotenv

📌 Funcionalidades:

Criar usuário

Listar todos os usuários

Buscar usuário por ID

Atualizar usuário

Deletar usuário

Renderização de dados com EJS

Este projeto foi desenvolvido com o objetivo de praticar conceitos de backend, estruturação de APIs e integração com banco de dados.

💻 Ideal para estudo de Node.js, APIs REST e Server Side Rendering com EJS.
